Welcome to Quiz App!

Features:
  • Quiz Solving: Users can engage in interactive quizzes covering various topics. Each quiz offers multiple-choice questions with instant feedback.
  • Quiz Creation: Registered users have the ability to create their own quizzes. They can customize quiz titles, questions, and answer options, allowing for a personalized quiz experience.
  • Leaderboard: The application includes a leaderboard feature displaying the top scores achieved by users across different quizzes. This provides a competitive element and encourages users to strive for high scores.
  • Shuffle Questions: To enhance user engagement and challenge, the app offers a feature to shuffle the order of questions within a quiz. This keeps the experience fresh and prevents users from memorizing question sequences.
